Hinweise:
  • Dies ist ein Beispiel-Seiten-Satz, der einen kompletten Shop für den PCG darstellt.
  • Dies könnte die Startseite der WebSite sein. Zur Illustration sind nicht alle Seiten komplett gleich. Diese Seiten können unter https://oekobox-online.de/v3/demo/<shopname> (als zip) oder via GitHub heruntergeladen werden.
    Auf den Seiten (ca. 10) finden sich Hinweise zu den dort wichtigsten Komponenten (insgesamt 12).
    Hier ist das Zusammenspiel der Komponenten, und hier das der Demo-Seiten visualisiert.
  • Doku der Komponenten und des CSS unter Ökobox-Online
  • Insgesamt wird versucht, hier so wenig wie möglich Code (HTML und CSS) zu verwenden, um dem anpassenden Designer soviel freie Hand wie möglich zulassen. Dadurch ist der Shop hier nicht so hübsch wie er sein kann ;-)
  • Komponenten sind zur Hervorhebung gestrichelt umrandet.
  • English beschriftete Links funktionieren nicht, die Funktionalität ist gerade im Aufbau.
  • Manche Komponenten können Elemente überall in der Seite Steuern, hier der Warenkorbwert von der Status-Komponente: 0
  • Hier gehts zur eingestellten Startseite des Systems (also zum bisherigen Shop). Diese Seite wird auch angesteuert, wenn der Browser nicht mit der Technologie des Shops klarkommt.
  • Im Admin-Bereich hinterlegte AGB-Seite.
Ticker-Komponente (unten)
  • Dient als Ticker und kann auch als Teaser (also für einen Artikel) konfiguriert werden
  • Aktuell muss die Warengruppe oder Rubrik oder der Artikel als Id angegeben werden , der Dargestellt werden soll. Da wird es noch weitere Optionen geben.
Navigations-Komponente (oben)
  • Bildet den Navigationsbaum ab, wie er im PCG eingestellt wurde. Sofern diese (neue) Einstellung im PCG noch nicht verwendet wird, wird ein an den bisherigen Shop angelehnter Standard verwendet.
